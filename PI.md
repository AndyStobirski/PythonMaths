# PI

Machin's formula is a mathematical trick used to quickly and precisely calculate the value of Pi (\(\pi \)). Created by astronomer John Machin in 1706, it connects Pi to simpler, smaller fractions using trigonometric functions. It can be expressed as:

$
\frac{\pi}{4}\approx 4arctan(\frac{1}{5}) - arctan(\frac{1}{239})
$
